# How to Cite CycloneNet

If you use CycloneNet in your research, please cite the following:

## Software Release

Ferreira, E. S. (2026). CycloneNet: A Reproducible Physics-Guided Pipeline for Forensic Rapid-Intensification Analysis — and an Honest Audit of What It Can and Cannot Do. GitHub. https://github.com/estefano-ferreira/cyclone-net

## BibTeX

```bibtex
@misc{cyclonenet2026,
  author = {Ferreira, Estefano Senhor},
  title = {CycloneNet: A Reproducible Physics-Guided Pipeline for Forensic Rapid-Intensification Analysis --- and an Honest Audit of What It Can and Cannot Do},
  year = {2026},
  publisher = {GitHub},
  journal = {GitHub repository},
  howpublished = {\url{https://github.com/estefano-ferreira/cyclone-net}}
}
```

> **Note:** the earlier title "CycloneNet V2: A Forensic Physics-Guided Deep Learning
> Framework for Atmospheric Singularity Mapping" is superseded and should not be cited
> (see `ERRATA.md`, item 5). The Zenodo record is updated to V3 with the revised title.
