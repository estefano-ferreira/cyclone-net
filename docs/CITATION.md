# How to Cite CycloneNet

If you use CycloneNet in your research, please cite the following:

## Software Release

Ferreira, E. S. (2026). CycloneNet: A Reproducible Pipeline for Forensic Rapid-Intensification Analysis — and an Honest Audit of What It Can and Cannot Do. GitHub. https://github.com/estefano-ferreira/cyclone-net

## BibTeX

```bibtex
@misc{cyclonenet2026,
  author = {Ferreira, Estefano Senhor},
  title = {CycloneNet: A Reproducible Pipeline for Forensic Rapid-Intensification Analysis --- and an Honest Audit of What It Can and Cannot Do},
  year = {2026},
  publisher = {GitHub},
  journal = {GitHub repository},
  howpublished = {\url{https://github.com/estefano-ferreira/cyclone-net}}
}
```

## Dataset

The dataset (v2 labels, CC BY 4.0) will carry its own Zenodo concept DOI —
citation entry to be added here when minted at the dataset release.

> **Note:** the earlier titles "CycloneNet V2: A Forensic Physics-Guided Deep
> Learning Framework for Atmospheric Singularity Mapping" and the interim
> "...Reproducible Physics-Guided Pipeline..." variant are superseded and
> should not be cited — the "physics-guided" label was retired on 2026-07-16
> (see `ERRATA.md` item 5 and `docs/hypothesis_registry.md`, H8). The Zenodo
> record is updated with the revised title.
